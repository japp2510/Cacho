# Cacho game in Python

The game consist of throwing the dice and full and introduce the values you got into your scoreboard.

Each player will be able to throw up to 3 times, a total of 5 dices,

There are 6 squares that represent each face of the die.
There are other 5 squares which must meet special condition (staires 1-5 or 2-6,full (3 dice facing one side and the other 2 other one), poker (4 dices facing same side)

The other 2 is having all 5 dices facing the same side.

The winner is the one who gathers more points.
